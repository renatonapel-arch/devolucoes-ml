🔄 O fluxo que você descreveu

1
Lista do que está voltando

2
Funcionário bipa a etiqueta

Código de barras, QR ou digita qualquer ID. O sistema testa todos os tipos de ID até achar.

Se o código não bater com nenhuma em trânsito, consulta a API do ML e traz mesmo assim.

4

Máximo de dados da devolução agrupados pra leitura rápida + ação de recebimento que alimenta o pipe.

🔑 Os IDs que a etiqueta pode trazer

  • 2000…
    Nº da venda (order/pack) — o número grande impresso
  • 552…
    Claim ID — a chave-raiz da devolução no ML
  • 47…
    Cód. envio Mercado Envios — o código de barras de cima
  • AP…BR
    Rastreio Correios (etiqueta kraft) — ou códigos de outras transportadoras
  • MLB…
    Cód. do anúncio — fallback extra
Como cada etiqueta usa um número diferente, a busca varre os 5 ao mesmo tempo. Bipou → achou, não importa a transportadora.
